### 벌크성 수정 주의점
- 설명
    - DB에다가 바로 JPQL을 날리기 때문에, DB에는 해당 수정이 반영됨
    - 하지만 영속성 컨텍스트에는 반영이 안됨
    - ⇒ 벌크성 수정 이후, 영속성 컨텍스트를 비워줘야함
- 방법
    - 직접 영속성 컨텍스트 비우기(test1)
        - em.flush()
        - em.clear()

- JPA 함수 사용(test2)
    - @Modifying(clearAutomatically = true)
        → 벌크성 수정 이후, 영속성 컨텍스트를 비워줌
