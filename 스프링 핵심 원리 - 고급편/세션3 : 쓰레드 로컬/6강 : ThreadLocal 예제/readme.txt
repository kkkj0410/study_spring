### <요약>
- ThreadLocal
    - 의미
        - 각 쓰레드가 싱글톤 객체 참조 시, 서로 독립된 공간을 할당받는 것
        - ⇒ 즉, A가 싱글톤 객체의 내용을 바꿔도, B는 그에 대한 영향을 받지 않음
    - ThreadLocal은 라이브러리로 지원함
---------------
- 동시성 문제를 해결하려고 한다.
    - 동시성 문제가 발생하는 로그 추적기의 변수 타입을 ThreadLocal로 사용한다

- 로그 추적기
    - nameStore를 ThreadLocal 타입으로 선언
    - ⇒ 쓰레드가 nameStore을 바꿀 시, 해당 쓰레드에게만 바꾼 내용이 반영됨
