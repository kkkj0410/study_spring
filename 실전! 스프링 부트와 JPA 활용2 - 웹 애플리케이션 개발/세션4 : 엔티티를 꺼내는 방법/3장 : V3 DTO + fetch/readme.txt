### 1. fetch
- 의미
    - + 개념
    - 테이블 조회 시, 다른 테이블도 가져오는 것
    - A 테이블에 B 테이블을 fetch 조인했다면 B 테이블도 가져옴
        - (B 테이블에 C 테이블이 들어있다면? → C 테이블은 프록시)

- fetch
    - Order 객체의 테이블을 가져오려 함
    - fetch o.member, o.delivery를 통해, 해당 테이블도 동시에 가져옴

- 출력
    - 과정
        - fetch 조인(테이블 조회)을 통해 테이블을 가져옴
        - 해당 테이블을 리스트 형태로 만듦
        - 해당 리스트를 DTO 형태로 재가공함
        - DTO 형태로 데이터 출력
